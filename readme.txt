Automatically appends the kwizzme want button to your posts.

== Description ==

This plugin is designed to help you make use of kwizzme affiliate program. It is intended for travel blogs.

Once you activate the plugin the kwizzme team will be notified of new posts that you write. Then a kwizzme team
member will read your post and prepare a kwizzit link.

The kwizzit link directs users who are interested in the described travel to kwizz creation form which will allow
them to create a kwizz. A kwizz is a travel wish a user has. This wishes are answered with individual offers by travel
agencies.

The button will appear only after the kwizzit link is created.

DISCLAIMER: No personal or tracking information is sent to kwizzme servers. Only the title, date and permalink of your
posts is sent.

== Installation ==

Install like any wordpress plugin, then go to "Kwizzme Button" menu page and enter the token you received from
kwizzme team.
